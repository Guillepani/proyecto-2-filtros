# Proyecto 2 – Tienda de filtros

Aplicación web tipo tienda online (con estilo Apple Store) desarrollada como segundo proyecto del bootcamp Full Stack.  
El objetivo principal del proyecto es trabajar la lógica en JavaScript, especialmente el manejo de arrays, objetos, filtros y renderizado dinámico del DOM.

# Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Git + GitHub

# Descripción del proyecto
La aplicación muestra un catálogo de productos cargados desde un array de objetos.  
El usuario puede filtrar los productos según distintos criterios y ver los resultados actualizados.
Todo el renderizado se realiza dinámicamente desde JavaScript.

# Conceptos trabajados
- Array de objetos
- Métodos de arrays (`filter`, `map`, `forEach`)
- Manipulación del DOM
- Gestión de eventos
- Estructuración básica de un proyecto frontend
